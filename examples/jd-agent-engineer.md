# Agent 开发工程师（高级）

## 职责
- 设计并实现面向生产环境的 LLM agent：工具调用、多步推理编排、上下文管理
- 搭建 agent 的评测体系（离线评测集 + LLM-as-judge + 线上指标）
- 优化 token 成本与延迟，支持多模型供应商切换

## 要求
- 熟悉至少一个 agent 框架/harness 的内部实现（如 pi、Claude Agent SDK、OpenAI Agents SDK）
- 理解 function calling、流式输出、结构化输出的底层协议
- 有 NLU/对话系统或 LLM 应用的实际落地经验
- 加分项：开源贡献、自研 agent 项目、评测/可观测性建设经验
